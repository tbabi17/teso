Contact Хувь хүн буюу иргэдийн харилцагчдын жагсаалт. Энэ хэсэгт хувь хүнийг харилцагчаар бүртгэх, хайх, түүнтэй холбоотой Асtivity -уудыг үүсгэх зэрэг үйлдлүүдлийг гүйцэтгэх боломжтой.

Views Харилцагчийн жагсаалтыг системээс тодорхойлсон эсвэл өөрийн хүссэн нөхцөлөөр харах боломжийг олгоно. Жишээ нь Prospect List гэвэл зөвхөн байгууллагын хувь дахь харилцагч болох магадлалтай иргэдийн жагсаалтыг харуулах гэх мэт. Create Personal View коммандаар өөрийн хүссэн байдлаар жагсаалтыг харах хэлбэр буюу шүлтүүрийг зохион байгуулах боломжтой.

New Шинэ хувь хүн бүхий харилцагчийг бүртгэх. New contact гэсэн цонх нээгдэх бөгөөд харилцагчийн талаарх үндсэн болон холбоо барих, бусад тайлбар гэсэн талбаруудыг оруулах хэрэгтэй. Овог нэр болон, утасны дугаарыг зайлшгүй оруулсан байх шаардлагатай. Хэрэв аливаа талбар улаанаар хүрээлэгдсэн байвал тухайн талбарыг заавал бөглөнө гэж ойлгоно. Level гэсэн талбарт тухайн харилцагчийн түвшин буюу байгууллагын жинхэнэ харилцагч мөн эсэхийг тодорхойлно. Жишээ нь Suspect гэвэл тухайн харилцагч одоогоор холбоо тогтоогүй зорилтод харилцагч, Prospect гэвэл тухайн харилцагчтай холбоо тогтоож байсан буюу магадлалтай харилцагч, Customer гэвэл аль хэдийн харилцагч болсонг заана. Type талбарт RETAIL - энгийн, VIP - онцгой харилцагч, EXPAT - гадаададын иргэн гэж зааж өгнө. Social Link талбарт тухайн хувь хүний хувийн веб хуудас, Facebook хуудас гэх мэт зааж өгч болно.

Expand Жагсаалтаас сонгосон харилцагчийн хувь дэлгэрэнгүй мэдээлэлийг дэлгэцийн баруун талд гаргана. Үүнд тухайн харилцагчийн Property буюу бусад талбарууд, хийж байсан Activity, өмнө нь гаргаж байсан Case - үүд гэх мэд. Property хэсэгт харилцагчийн мэдээлэлийг харгалзах талбараар нь сонгон засах, мэдээлэл нэмэх боложмтой бөгөөд Commit товчоор өөрчлөлтөө хадгална.

Delete Жагсаалтаас сонгосон харилцагчийг устгана. Зөвхөн өөрийн оруулсан харилцагчийг устгах боломжтой. Мөн тухайн харилцагчийг устгаснаар түүнтэй холбоотой бусад түүхүүд устна гэдгийг анхаарах хэрэгтэй.

Merge Жагсаалтаас сонгосон ижил гэж үзсэн 2 харилцагчийн мэдээлэлийг нэгтгэж нэг харилцагч болгон бүртгэх зориулалттай. Зөвхөн ахлах болон түүнээс зэрэгтэй ажилтан хийж гүйцэтгэх боломжтой. 2 харилцагчийг Ctrl + Mouse Left Click дарж зэрэг сонгож болно. Merge цонх онд сонгосон 2 харилцагчийн мэдээлэл баруун зүүн талд гарах бөгөөд харгалзах өгөгдөлүүдийг нэгтгэх замаар бүртгэнэ. Merge хийсний дараа тухайн 2 харилцагчийн бүхий л түүхүүд нэгдэх болно.

Import Тодорхой Excel загварын дагуу бэлдсэн олон харилцагчийн мэдээлэлийг нэг коммандаар оруулах.

Export Тухайн харагдаж буй харилцагчийн жагсаалтыг Excel-руу гаргах.

Activity Сонгосон харилцагчийн хувьд Notes, Task, Appointment, Call, Email хийсэн log-уудыг бүртгэх, өмнөх Log-уудыг харах.

Case Сонгосон харилцагчийн хувьд Case-үүдийн жагсаалт болон шинэ Case үүсгэх гэх мэт. Тухайн харилцагч дээр гомдол санал, мэдээлэл хүргэсэн, шийдвэрлэсэн зэргийг бүртгэнэ.

Deal Сонгосон харилцагчийн хувьд Deal-үүдийн жагсаалт болон шинэ Deal үүсгэх гэх мэт. Харилцагчтай ямар нэгэн борлуулалтын үйл ажиллагааг эхлүүлэх үйлдлийг үүгээр хийнэ.
