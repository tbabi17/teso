Агуулахын жагсаалт Агуулахын жагсаалтыг харуулах хэсэг. Агуулах шинээр нэмэх, устгах, засах үйлдлүүдийг энд хийнэ.

Нэмэх Шинээр агуулах үүсгэх.

Засах Сонгосон агуулахын мэдээллийг засварлах. Жагсаалтаас агуулахын сонгох ётой.

Устгах Сонгосон агуулахыг устгах. Уг үйлдлийг хийхэд агуулахын бүх мэдээлэл устах болно.
