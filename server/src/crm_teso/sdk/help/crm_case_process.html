Cases Ажилтан таньтай холбоотой үүссэн Case-үүдийн жагаалт энд харагдана. Та өөрйин үүсгэсэн болон таньд оноогдож ирсэн бүхий л Case-үүдийг эндээс хөтлөж, дараачын шат руу ахиулах, улмаар Case-ыг шийдвэрлэх хүртлэх бүх процессыг хөтлөнө. Тухай Case-ыг сонгосоноор баруун талд Stage нэртэй цонх гарах ба уг нь цонхонд Case-ын одоогийн үйл явц, Activity буюу Case - тэй холбоотой хийсэн Activity /утсаар ярьсан, майл явуулсан/ үүсгэх, ямар бүтээгдэхүүн үйлчилгээний хувьд гомдол, санал гаргасан, гомдол саналыг шийдвэрлэлтийн процесс буюу дамжлагыг харах гэх мэт.

Delete Сонгосон Case-ыг устгах. Case - ыг устгавал түүнтэй холбоотой үүссэн Activity /Phone Call, Email, Appointment гэх мэт/ бүх Log-ууд хамтдаа устана.


Stage

Next Stage Тухайн Case дараачийн шатруу оруулах комманд. Identify -> Research -> Resolve гэсэн 3 үе шаттай. Identify шат буюу Case - ын эхний төлөв. Research - шийдвэрлэлтийн үе шат. Энэ үе шатанд Case - ыг хариуцсан ажилтан руу шилжүүлэх гэх мэт үйлдлийг хийх боломжтой. Case - тай холбоотой утсаар ярьсан, уулзсан, майл явуулсан Log - уудыг Activity хэсэгт бүртгэх зэрэг үйлдлүүдийг хийхэд тохиромжтой. Resolve - буюу Case шийдвэрлэх үе шат.

Assign Case - ын өөр ажилтанд оноох буюу шилжүүлэх. Шилжүүлсэний дараа танд тухайн Case харагдахгүй болно. Шилжүүлэхийн тулд тухайн ажилтаны хэрэглэгчийн нэрийг бичиж илгээнэ.

Resolve case Case - ыг шууд хаах буюу шийдвэрлэх.

Cancel case Case - ыг шууд хаах буюу цуцлах.
