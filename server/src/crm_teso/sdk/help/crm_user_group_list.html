Deal Deal буюу харилцагчтай хийх борлуулалтын процессыг эхлүүлэх үе шатыг энд үүсгэнэ. Энд тухайн харилцагчтай өмнө үүсгэж байсан Deal-үүд тэдгээрийн биелэлт, ямар түвшинд байгаа зэрэг мэдээлэлийг харах боломжтой. Энд үүссэн Deal нь DEALS гэсэн том гарчигтай цонхонд бусад харилцагчийн үүссэн Deal-үүдтэй нийлж, тэндээс тухайн Deal-ыг цааш үргэжлүүлэх үе шатыг хөтлөнө. DEALS цонх нь тантай холбоотой Deal-үүдийн жагсаалтууд байх юм.
