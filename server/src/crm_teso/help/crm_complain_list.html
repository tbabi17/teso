Case Харилцагчтай холбоотой санал хүсэлт(гомдол), чухал шийдвэрлэх зүйлс зэргийг энд бүртгэгдэнэ.

New Сонгогдсон харилцагчийн хувьд шинээр Case үүсгэх. Status - Case -ын төлөв, Case Reason - Case - ын шалтгаан буюу гарчиг, Stage - үе шат indentify буюу тодорхойлох үе шат, research - буюу шийдвэрлэх үе шат, resolve буюу амжилттай шийдвэрлэсэн гэсэн шатуудыг заана. Origin - ямар шугамаар орж ирсэн, Type - Case - ын ангилал /question, problem, feature request/. Close date - Case - ыг хаах огноо. Хаагдах огноо хэтэрсэн тохиолдолд таньд анхааруулга ирэх болно.

Expand Тухайн сонгосон Case - ын дэлгэрэнгүйг харах, засварлах.

Delete Тухайн сонгосон Case - ийг устгах. Case - ынг устгаснаар тухайн Case - тай холбоотой хийгдсэн Activity, түүхүүд устгах болно.

Calendar Google Account-аар дамжуулан Google Calendar - т оруулах. Ингэснээр таны смартфон дээр тухайн Case-ыг сануулах, аларм өгөх боломжтой.
